Политика конфиденциальности

BeePuzzle · Обновлено 08.07.2026

Настоящая Политика описывает какие данные собираются при использовании игры BeePuzzle («Пчелкины пазлы»), как они используются и с кем могут делиться. Устанавливая и используя приложение, вы соглашаетесь с условиями этой политики.

1. Разработчик

Приложение разработано физическим лицом Пуцык Степан Николаевич (статус: самозанятый, НПД).

Контакт для вопросов и запросов: putsyk.stepan@yandex.ru.

2. Какие данные собираются

Приложение НЕ собирает и не запрашивает у пользователя персональные данные (имя, номер телефона, точное местоположение, фото, контакты).

Приложение автоматически передаёт следующую техническую информацию через сторонние SDK:

  • Рекламный идентификатор устройства (Advertising ID) — для показа релевантной рекламы.
  • Модель устройства, версия ОС, язык системы — для аналитики стабильности и подбора рекламы.
  • Данные о сессии — время старта/завершения игры, страна по IP, тип соединения.
  • События внутри игры — прохождение уровней, использование бустеров, показы рекламы (обезличенно).
  • Данные о крашах — стек ошибок для исправления багов.

3. Локальное хранение прогресса

Прогресс в игре (пройденные уровни, собранные монеты, настройки) сохраняется локально на вашем устройстве через стандартный механизм PlayerPrefs Unity. На серверы разработчика эти данные не передаются.

При удалении приложения или очистке данных прогресс безвозвратно теряется — облачное сохранение пока не реализовано.

4. Используемые SDK третьих сторон

Приложение использует следующие сторонние сервисы:

  • Yandex Mobile Ads — показ рекламы. Политика: yandex.ru/legal/mobileads_termsofuse
  • Yandex AppMetrica — аналитика использования. Политика: yandex.ru/legal/appmetrica_agreement
  • Unity Analytics (при наличии) — техническая аналитика движка Unity.

5. Реклама

В приложении отображается реклама (полноэкранная и с вознаграждением). Она предоставляется рекламной сетью Yandex Mobile Ads и может использовать рекламный идентификатор устройства для персонализации.

Вы можете сбросить рекламный ID или отключить персонализацию в системных настройках вашего устройства (Настройки → Google / Конфиденциальность → Реклама).

6. Возраст пользователей

Игра рассчитана на аудиторию 0+ / 3+ и не содержит контента для взрослых. Мы не собираем осознанно данные детей до 13 лет — только обезличенную техническую информацию, общую для всех пользователей.

7. Ваши права

Вы имеете право:

  • Запросить какие данные о вас были собраны.
  • Запросить удаление ваших данных из систем аналитики.
  • Отказаться от персонализированной рекламы через системные настройки устройства.

Запросы направляйте на putsyk.stepan@yandex.ru — ответ в течение 30 дней.

8. Изменения политики

Мы можем обновлять эту политику. Актуальная версия всегда доступна по адресу stepanputsyk.github.io/privacy.html. Существенные изменения будут отражены в дате «Обновлено» в шапке.
